# Buddy Product Vision

## Mission

Buddy helps people learn more effectively by combining artificial intelligence with collaborative learning.

Buddy is not an AI chatbot.

Buddy is an intelligent study companion.

---

# Core Philosophy

Learning improves when people:

- Think
- Explain
- Practice
- Collaborate

instead of simply receiving answers.

---

# Core Pillars

## 1. AI Study Companion

Buddy acts as a long-term learning partner.

Responsibilities:

- Study planning
- Guided learning
- Quizzes
- Flashcards
- Revision
- Progress tracking
- Personalized explanations
- Long-term learning memory

---

## 2. Human Collaboration

Buddy connects learners.

Features include:

- Study partners
- Study groups
- Shared goals
- Shared notes
- Chat
- Video rooms
- Pair programming

---

## 3. Active Learning

Buddy encourages learning using modern educational techniques.

Methods include:

- Socratic Questioning
- Retrieval Practice
- Guided Discovery
- Adaptive Teaching
- Reflection
- Spaced Repetition

Buddy should avoid giving answers immediately whenever possible.

---

## 4. AI Productivity

Buddy teaches users how to use AI effectively.

Features:

- Prompt Refinement
- Prompt Templates
- Prompt Library
- Prompt Recommendations
- Prompt History

---

# Current MVP

Current focus:

One learner

+

One AI companion

Core features:

- Authentication
- User profiles
- Study sessions
- Progress tracking
- AI chat
- Learning memory
- Prompt refinement

---

# Long-Term Vision

Buddy should become an intelligent learning ecosystem capable of supporting individual learners, study groups, classrooms, and professional teams.
